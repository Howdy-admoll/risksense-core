QUICK FILE DESCRIPTIONS:
========================

Core Model (risksense/model.py — 770 lines):
  - RiskSenseModel class: Main Mamdani FIS implementation
  - Input variables: annual_income, debt_to_income, credit_score, employment_stability
  - Output variable: risk_score (0-100) with categories
  - 32+ fuzzy rules encoding financial heuristics
  - Triangular membership functions
  - Centroid defuzzification
  - Input validation & batch processing
  - create_model() factory function

Test Profiles (risksense/profiles.py):
  - 8 realistic borrower archetypes
  - Coverage: Low, Medium, High risk categories
  - Based on Nigerian fintech market segmentation
  - get_profiles() — return all profiles
  - get_profile_by_name() — retrieve single profile
  - get_profiles_by_category() — filter by risk

Test Suite (tests/test_model.py — 350+ lines):
  - TestModelInitialization: FIS setup validation
  - TestInputValidation: Bounds checking
  - TestAllProfiles: All 8 profiles score correctly
  - TestRiskCategorization: Low/Medium/High boundaries
  - TestBatchProcessing: Multiple borrower scoring
  - TestSensitivityAnalysis: Parameter impact analysis
  - TestEdgeCases: Extreme input combinations

CLI (risksense/cli.py — 350+ lines):
  - score: Score single borrower
  - batch: Batch CSV processing
  - analyze: Sensitivity analysis on parameters
  - inspect: Model structure introspection
  - Multiple output formats (JSON, CSV, table)

API (risksense/api.py — 400+ lines):
  - Flask microservice with 5 endpoints
  - /api/health: Health check
  - /api/model/info: Model specification
  - /api/profiles: List test profiles
  - /api/score: Single borrower scoring
  - /api/batch: Batch scoring
  - JSON request/response
  - Error handling & validation

Visualization (risksense/visualization.py — 300+ lines):
  - plot_membership_functions(): Fuzzy set charts
  - plot_profile_risks(): Bar charts of profile scores
  - plot_sensitivity_heatmap(): 2D parameter interaction heatmaps
  - Matplotlib-based with high-quality output

GETTING STARTED:
================

1. Install:
   pip install -r requirements.txt

2. Test:
   pytest tests/test_model.py -v

3. CLI:
   python -m risksense.cli score --income 2.5 --dti 0.4 --credit 75 --stability 8.0

4. API:
   python -m risksense.api

5. Examples:
   python examples/01_individual_scoring.py
   python examples/02_batch_portfolio_analysis.py

6. Docker:
   docker build -t risksense:latest .
   docker run -p 5000:5000 risksense:latest
